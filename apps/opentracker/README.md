# OpenTracker

OpenTracker is an open-source website for tracking workouts, mainly aimed at
powerlifting specifically. It provides the following features:

* Basic user account control
* Logging and graphing bodyweight over time
* Recording workout information
	* Viewing 1-RM progression for each lift
	* Providing a summary of workouts that have been recorded

The frontend for the website is written in React and the backend is written in
Rust. It can be viewed [here](tracker.blackboards.pl).
